# Netflix Clone 🎬

This is a front-end clone of the Netflix website built using HTML, CSS, and JavaScript. The project focuses on replicating the UI and user experience of Netflix with a responsive design.

## 🚀 Features
- Responsive design for mobile, tablet, and desktop
- Attractive homepage UI similar to Netflix
- Navigation bar with hover effects
- Movie/TV show sections with thumbnails
- Banner section with featured content

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
- index.html
- style.css
- script.js
- images/
